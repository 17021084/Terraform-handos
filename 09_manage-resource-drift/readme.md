# Refesh


### Replace resource (taint)

Thay force thế resource  được chỉ định 


```bash
terrafrom apply -replace=<resource address>

terrafrom apply -replace=aws_resource.ec2_server

# tự động replace resource. ( xoá đi và tạo lại )
```
